### Uploading a file during a test

You can click the paperclip icon on the header of the test instructions input to open the file selector. 

![Screenshot 2025-07-18 at 3.46.46â€¯PM.png](/images/Screenshot2025-07-18at3.46.46PM.png)

Once you've picked a file, you can simply say `upload file` as part of any test prompt and Docket will automatically place that file in the relevant input component on your webpage. For example, you might prompt Docket with "upload file into the profile picture input".
